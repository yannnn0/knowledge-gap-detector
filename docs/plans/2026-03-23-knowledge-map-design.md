# 知识地图网站设计文档

## 项目概述

- **项目名称**: 我的知识地图
- **类型**: 交互式单页应用 (React)
- **核心功能**: 帮助用户（LeanFaella）通过分析微博内容，发现自己的知识结构、能力长短板、知识空白，并推荐学习路径
- **目标用户**: LeanFaella 本人，研究生毕业 gap 期，需要发现知识盲区

## 功能规格

### 1. 数据输入
- **粘贴内容**: 文本输入框，支持粘贴微博文本
- **上传文件**: 支持 TXT/MD/JSON 文件上传
- **解析逻辑**: 提取关键词、主题词，分类到各个知识维度

### 2. 雷达图 + 空白检测
- **维度**:
  - 职业规划
  - 商业/变现
  - 财务管理
  - 学术研究
  - 情绪心理
  - 人际关系
  - 创意技能
- **空白检测**: 统计各维度话题频率，低频领域标记为空白
- **可视化**: 交互式雷达图，点击维度显示详情

### 3. 节点网络图
- **节点类型**: 知识点、主题、关键词
- **关联关系**: 主题间的关联度
- **交互**: 点击展开、缩放、拖拽
- **高亮**: 空白区域节点用特殊颜色标识

### 4. 学习路径推荐
- 根据空白领域推荐学习方向
- 推荐具体资源/方向

## 技术栈
- React + TypeScript
- Tailwind CSS
- shadcn/ui 组件
- Recharts (雷达图)
- react-force-graph (节点网络图)

## 视觉风格
- 清新、个性化、有创意
- 浅色背景 + 柔和渐变
- 流畅动画
- 响应式设计
